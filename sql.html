Understanding SQL

SQL is the more traditional approach to a database. An SQL database stores data in a table, which is great for handling structured data, in other words, data which is very predictable and organized. An example of structured data may be a search engine, which requires organized and predictable data to provide the proper results. An alternative to SQL, NoSQL, has gained popularity with the advent of gaming and social networks as it allows for more flexibility, such as allowing the addition of new properties on the fly, as well as for better handling on non-structured data, or data that is likely to be manipulated and changed frequently.
