Understanding Git Merges

In order to understand merges in git, a useful resource is the Git documentation website, specifically the section regarding git merges. This resource provides an overview of the git merge command, as well as how to handle various processes involving git merge. Processes such as resolving conflicts. This resource does a good job at addressing the commands that will be useful in addressing the various scenarios which may arise due to a git merge.

On my mac, it would appear as though a text editor is opened within the terminal upon merging. To resolve a conflict, I must navigate to the file which has an error to be resolved and execute a git status command.
